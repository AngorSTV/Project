# Roadmap

Документ описывает план поставок и приоритеты. Поддерживать в актуальном состоянии по итогам релизов и ключевых решений (ADR).

## 2025 Q4 → 2026 Q1 (ближайший фокус)
- MVP анонимного сборщика ПК: подбор комплектующих → анализ → рекомендации по 4 сценариям.
- Публичные API справочников комплектующих: безопасность, лимиты, кеширование.
- Уточнение формата результатов анализа (AnalyzeResponseDto) и текстов рекомендаций.
- Стабилизация пайплайнов и окружений (dev/stage/beta/prod), формализация деплоя.

## 2026 Q2 (наращивание ценности)
- Расширение модели рекомендаций: учёт БП, накопителей, корпуса/охлаждения, совместимости.
- Витрина лучших конфигураций под бюджет/сценарии.
- Монетизация: подписки + аффилиат‑редиректы с сегментацией free/premium.

## Метрики успеха (черновик)
- Конверсия из подбора в анализ
- Доля повторных пользователей
- CTR по рекомендациям/редиректам
- NPS/оценка полезности рекомендаций
