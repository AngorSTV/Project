# Snapshot

Этот файл перезаписывается автоматически в CI перед сборкой MkDocs.

Если ты смотришь **сайт (GitHub Pages)** — ты видишь актуальную версию.
Если ты смотришь **raw в репозитории** — здесь может быть placeholder.

Для нового чата по проекту: открывай страницу Snapshot на Pages и ориентируйся на строку `Generated:`.
